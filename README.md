# SmartTask Hub Docker

A production-style Django application built to learn and demonstrate Docker, Docker Compose, PostgreSQL, Redis, Celery, Nginx, persistent storage, networking, and scalable deployment practices.

## Project Goal

This project is part of my Data Engineering and Backend Engineering learning journey.

The objective is to build a real-world containerized application while mastering:

- Docker
- Docker Compose
- PostgreSQL
- Redis
- Celery
- Nginx
- Volumes
- Networks
- Health Checks
- Container Security
- Scaling
- Production Deployment
